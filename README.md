# Smart Automated Bottling Production Line

## Objective
Design and implement a PLC-based automated bottling line with four sequential stations—**Filling, Capping, Labelling, and Inspection**—using sequential control, safety interlocks, pneumatic actuators, conveyor automation, quality inspection, and production counting.

## Overview
A PLC-controlled bottling process where bottles move through four automated stations via a conveyor. The program demonstrates industrial sequencing, timer-based operations, equipment interlocks, safety logic, and production monitoring.

## Features
- Emergency Stop (E-Stop)
- Conveyor motor feedback
- Air pressure interlock
- Tank level interlock
- One-shot cycle start
- Automatic filling
- Pneumatic clamping
- Automatic capping
- Automatic labelling
- Quality inspection
- Good & reject bottle counting
- Timer-based sequencing
- Station interlocking

## PLC Instructions Used
- XIC / XIO
- OTE
- OTL / OTU
- ONS
- TON
- CTU / CTD
- GEQ
- MOV
- Internal Memory Bits
